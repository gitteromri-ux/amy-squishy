## צעד 1 — לרשום את הדומיין (פעם אחת, ~40 ש"ח לשנה)

בעמוד הרישום:
1. הקלד: `amysquishy.co.il`
2. אם פנוי — לחץ "רכוש / הזמן".
3. שלם. זהו, הדומיין שלך.

> אם amysquishy.co.il תפוס — תגיד לי ואציע חלופות פנויות (למשל amy-squishy.co.il, amysquishy.co.il).

## צעד 2 — להפנות את הדומיין ל-Cloudflare (קופי-פייסט)

**מחק** מה שכתוב שם, והדבק את שתי השורות האלה במקום:

```
NS  →  (אני אתן לך את שתי הכתובות המדויקות אחרי שתפתח את הדומיין ב-Cloudflare — שורה אחת לכל שדה)
```

> אל תדאג מזה עכשיו. ברגע שתגיד לי "רשמתי", אני פותח את הדומיין ב-Cloudflare שלך ושולח לך בדיוק את 2 השורות להעתיק.
